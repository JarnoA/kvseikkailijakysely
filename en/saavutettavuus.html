Accessibility Statement

Updated: 23.3.2026

This accessibility statement applies to the website kvseikkailijakysely.fi. The website is subject to the Act on the Provision of Digital Services, which requires that public web services must be accessible.

Preparation of the statement

This statement was prepared based on self-assessment on 23.3.2026. The evaluation has taken into account the technical implementation and user interface of the quiz application.

1. Status of accessibility of the digital service

This web service partially meets the accessibility requirements set by law (WCAG 2.1, level AA) due to the deficiencies mentioned below.

2. Non-accessible content

The following accessibility deficiencies have been observed in the service, which we aim to correct or for which an alternative method is provided:

  • Use of QR code on the result page (WCAG 1.1.1 Non-text Content): On the test result page (for event use), a QR code is used to share the result. Reading the QR code requires vision and the use of another device, and it is not as such accessible to screen reader users. Alternative method: In addition to the QR code, the result page has text-based and screen reader accessible buttons ("Take the test yourself" or social media sharing buttons) that serve the same purpose without scanning a QR code.
  • Dynamic view transitions (WCAG 4.1.2 Name, Role, Value / WCAG 2.4.3 Focus Order): The quiz is implemented as a dynamic application where the transition from one question to another occurs without reloading the page. In keyboard and screen reader use, there may be delays or inaccuracies in the focus transition to a new question with certain browser and screen reader combinations.
  • Text alternatives for character illustrations (WCAG 1.1.1 Non-text Content): The illustrations used on the result page may contain visual details that are not perfectly described in the alt texts, although the result itself (character name and description) can be read as text.

3. Feedback and contact information

Did you notice an accessibility deficiency in our digital service? Let us know and we will do our best to correct it.

Email: kirjaamo@oph.fi

4. Enforcement procedure

If you notice accessibility problems on the site, first provide feedback to the site administrator (Finnish National Agency for Education). It may take 14 days to receive a reply. If you are not satisfied with the answer you received or do not receive a reply at all, you can make a report to the Regional State Administrative Agency for Southern Finland.
